# Контактная книга (Contact Book)
Консольное приложение для управления списком контактов. Демонстрирует практическое применение Java Collections Framework (List, Set, Map).

## Функции
*   Добавление/удаление/поиск контактов
*   Группировка контактов (Семья, Друзья, Работа, Другое)
*   Валидация ввода и обработка ошибок
*   Удобное консольное меню

## Структура проекта
*   `Contact` - класс сущности контакта
*   `ContactBook` - основной класс с бизнес-логикой
*   `Main` - точка входа с пользовательским интерфейсом
*   `ErrorMessages` - перечисление сообщений об ошибках
